自行制作唠铁APP指南

面向普通人的实用教程

提示：下述两种方法均免费，但可能需要忍受广告等。

【第一种方法】使用 HBuilderX 封装唠铁APP

这是比较专业的方法。

【一、环境准备】

  1. 下载安装：访问 HBuilderX 官网（www.dcloud.io）下载并安装最新版本。
  2. 账号注册：注册 DCloud 账号，并在 HBuilderX 中登录。
  3. 实名认证：若使用云打包功能，需在 DCloud 开发者中心完成实名认证。

【二、创建项目】

  1. 启动 HBuilderX，点击菜单栏 文件 > 新建 > 项目。
  2. 选择 5+ App 模板，选择 默认模板。
  3. 输入项目名称，选择存储路径，点击 创建。

【三、配置 manifest.json】

  1. 在项目根目录找到 manifest.json 文件并打开。
  2. 【基础配置】：设置 App 名称、标识（Package Name，需唯一）、版本号。一般使用默认即可。
    在 应用入口页面(首页)地址 输入打包网址：
    https://www.holomind.com.cn/litalk/chat_robot.html?uf=您的名字拼音
    其中【您的名字拼音】是与阿色给您的唠铁网址（https://www.holomind.com.cn/litalk/您的名字拼音）中相同的。形如：
    https://www.holomind.com.cn/litalk/chat_robot.html?uf=wanglaoliu
  3. 【图标配置】：在"自动生成图标"项中，指定一个图片（只认.png，且尺寸大于200*200）；然后按"自动生成所有图标并替换"键。
  4. 【模块配置】：在"模块配置"中，去掉"Contact(通讯录)、Geolocation(定位)"选项。
  5. 【权限配置】：在"权限配置"中，找到
    <uses-permission android:name="android.permission.READ_CONTACTS"/>，
    删掉它。

【四、打包 APK 文件】

  1. 点击菜单栏 发行 > App-云打包。
  2. 选择平台：勾选 Android。
  3. 证书配置：选择 使用云端证书。
  4. 打包类型：选择 打正式包。
  5. 广告联盟：去掉所有选择。
  6. 打包方式：选择 快速安心打包。
  7. 点击 打包键，等待云端编译完成（大约排队10分钟），过程中询问都点"继续打包"。

【五、获取、安装、分发 App 安装包】

  1. 打包完成后，.apk 文件会保存在本地电脑中，HBuilderX 将在底部显示文件夹。
  2. 找到该 .apk 文件，它就是 App 安装文件包。
  3. 将 .apk 文件传输至 Android 设备进行安装测试。
  4. 测试没问题的 .apk 即可分享给朋友们了，也应上传至唠铁服务器。
    如果测试有问题，可尝试清除原app的数据，或删除旧的app重新安装。
    经验：可以使用微信的收藏功能把你分发的 .apk 文件或网址保存，方便下次分发。
    * 安装和分发唠铁的技巧：点击查看
注：如需要帮助，可联系阿色，但阿色鼓励你自己先试试，挺有意思的。

【第二种方法】使用网页版打包工具 WebIntoApp 封装唠铁APP

这是比较简单的方法，有广告。

  1. 进入网站：https://www.webintoapp.com/

    教程：点击网站上的 Watch Video

  2. 免费注册账号：网页右上角 Author/Register
  3. 回到网站首页，点击 GET STARTED 开始；
  4. 在 App Maker 表单中填写：
    • 左边 URL: 填写打包网址：
      https://www.holomind.com.cn/litalk/chat_robot.html?uf=您的名字拼音
      其中【您的名字拼音】是与阿色给您的唠铁网址（https://www.holomind.com.cn/litalk/您的名字拼音）中相同的。例如：
      https://www.holomind.com.cn/litalk/chat_robot.html?uf=wanglaoliu
    • 右边 Extras：点 Set Icon 设置 logo。只允许使用 128px 的 PNG 文件，需要事先处理好。
  5. 点击 Next；
  6. 填 App Name（你想给你的 App 定的名字），例如：xiaoxiao，中文不允许；
    填 Package Name（App 文件包名字），例如 litalk.xiaoxiao.android；
  7. 点击 Next；
  8. 点击 Make App；需要密码。
  9. 稍等，在 Apps List 下面将列出已经做好的 App；点击"云"图标，再点 Download | FREE，再点 Download All Files(ZIP) 下载；
  10. 找到下载的文件，双击打开，在 android 文件夹中找到 app-release.apk，这个就是编译好的 App 安装文件包，把它复制保存即可分发上传了。
